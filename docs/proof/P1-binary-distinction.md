# P1：二元区分命题

## 命题陈述

**命题 P1**：任何区分的最小形式是二元的。

## 形式表述

设D是一个区分操作，则存在二元划分$\{A, B\}$使得：
$$
D(x) = \begin{cases}
A & \text{如果 } x \text{ 具有性质 } P \\
B & \text{如果 } x \text{ 不具有性质 } P
\end{cases}
$$
## 证明

### 逻辑基础

**步骤1：区分的定义**
区分意味着识别不同：
- 给定对象x
- 判断x是否具有某种性质P
- 结果必须是明确的

**步骤2：排中律**
在经典逻辑中，对于任何命题P：
- P或者为真
- P或者为假
- 没有第三种可能

**步骤3：二元划分**
因此任何区分都产生二元划分：
- 类A：$\{x | P(x)为真\}$
- 类B：$\{x | P(x)为假\}$

### 信息论角度

**步骤4：信息量**
最小的区分提供1比特信息：
- 0：属于类$A$
- 1：属于类$B$

**步骤5：最优性**
使用更多符号是浪费：
- 0, 1, 2：第三个符号冗余
- 可以用组合表示：2 = 10

### 本体论分析

**步骤6：存在的基本性质**
最根本的区分是：
- 存在 vs 不存在
- 是 vs 非是
- 自己 vs 非自己

这些都是二元的。∎

## 推论

1. **逻辑基础**：二值逻辑的合理性
2. **编码效率**：二进制编码的最优性
3. **认知基础**：思维的二元结构
